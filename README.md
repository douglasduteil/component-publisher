## Goal

```
+ git repo
  + master (src) branch // build directory is ignored
    - tag: src1.0.x
    - tag: src1.1.x
  + gh-pages (doc) branch
  + bower (build) branch // build directory is committed
    - tag: v1.0.x
    - tag: v1.1.x
  + bower-subcomponent (like angular-ui-event) branch
    - tag: ui-event-1.0.x
    - tag: ui-event-1.1.x
```
